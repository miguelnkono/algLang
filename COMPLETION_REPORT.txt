PREREQUISITES:

  • Maven 3.6+ (Check: mvn -version)
  • GraalVM optional (only for native builds)
BUILD & RUN:
1. Build with tests:

2. Run native executable:

   $ mvn clean package -Pnative-optimized
5. Run tests only:
   $ mvn test
